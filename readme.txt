Inserts a search field bar in Wordpress posts, pages and widgets. Your website visitors can search an Excel or .csv database that was uploadedor selected in the Media Gallery. Quick 2-step configuration and get your shortcode to place in posts, pages and widgets. Get it working in one minute.


== Description ==
Inserts a search field bar in Wordpress posts, pages and widgets so website visitors can search an Excel or .csv database.


== Installation ==
1. Search for **Search Excel csv** at the Wordpress Add Plugin dashboard
2. Click to install it
3. Activate it
4. In Installed Plugins, click in the plugin Settings link 

Alternatively:
1. Go to https://wordpress.org/plugins/search-excel-csv/
2. Download the plugin
3. Upload it through the Upload button at the Wordpress Add Plugin dashboard.
4. Activate the plugin through the "Plugins" menu in WordPress.

== Utilisation ===
At the plugin dashboard:
a. Upload your Excel or CSV file or Select it from the Media dashboard, 
b. Click the Generate Code button. It will copy automatically to the clipboard.
c. Paste the shortcode in your website page, post or widget.
